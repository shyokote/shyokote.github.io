    Delete Kernel

    Ubuntu

    稼働中のカーネルバージョンの確認 

    uname -r

    インストール済みカーネルのリストを確認 

    dpkg --get-selections | grep linux-

    カーネルの削除 (例として3.13.0-65, 3.13.0-67, 3.13.0-71を削除する場合) 

    sudo apt autoremove --purge linux-{headers,image-extra,image}-3.13.0-{65,67,71}
