    Zabbix WebUI Admin Password Change

    • ZabbixのWebUIログインパスワードを忘れてしまった。
    • ldap連携をしていなくて、Zabbixデータベースのユーザー認証である
    • 他にログインをできる管理者ユーザーがいない状態である

    概要

    • Zabbix サーバーのデータベース (ここではMySQL) にログインをして直接変更をする
    • 登録するパスワードは事前にbcriptでハッシュ化する (ここではデフォルトの zabbix とする)

    対処

    zabbixデータベースに接続 

    $ sudo mysql
mysql>  use zabbix;

    変更したいユーザーのIDを確認する 

    mysql> select userid, username from users;

+--------+----------+
| userid | username |
+--------+----------+
|      1 | admin    |
|      2 | guest    |
+--------+----------+
2 rows in set (0.00 sec)

    Passwordをzabbixにする (デフォルトにする)。 パスワード文字列をbcriptでハッシュしたものを使う 

    mysql> update zabbix.users set passwd='$2a$08$Da5nMv6HAVy/7FDjJHS.Lu0QQGg9SBTGorAMmjyKl.0Ohpgfl9tnS' where userid=‘1’;
mysql> flush privileges;

    確認

    Zabbix WebUIからログインをする。

    Note

    • zabbixのパスワードはWebUIからはハッシュ化されてデータベースに保存される
    • zabbixのデータベースにはハッシュ化したパスワードを格納する必要がある
    • zabbix 6.2からMD5からbcriptに変更された
